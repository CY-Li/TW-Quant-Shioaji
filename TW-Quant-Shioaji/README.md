# 🇹🇼 TW-Quant-Shioaji: 台股全自動量化交易系統

本專案旨在利用永豐金 (SinoPac) Shioaji API，打造一個針對台股市場的自動化交易系統。包含即時行情監控、策略回測、資金管理以及自動化下單功能。

## 📅 開發路線圖 (Roadmap)

### 第一階段：基礎建設與身份驗證 (Current)
- [ ] 建立專案結構。
- [ ] 實作 Shioaji API 登入與憑證驗證邏輯。
- [ ] 確保環境連線穩定性。

### 第二階段：市場數據引擎
- [ ] 串接台股即時 Tick 數據與 K 線數據。
- [ ] 建立歷史數據抓取工具，用於回測。

### 第三階段：策略移植與優化
- [ ] 將 BullPS 技術反轉邏輯移植至台股（如 RSI + 成交量過濾）。
- [ ] 考慮台股特性：10% 漲跌幅、逐筆交易、零股交易邏輯。

### 第四階段：回測與績效分析
- [ ] 建立回測引擎，支援多標的同時模擬。
- [ ] 產出績效報告：勝率、ROI、最大回撤 (MDD)。

### 第五階段：全自動交易執行
- [ ] 實作下單模組（API 下單、撤單、改價）。
- [ ] 建立即時風控與保證金監控。
